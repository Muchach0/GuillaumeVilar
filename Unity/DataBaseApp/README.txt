insertdatabase.apk is a simple android application which will send some information
filled by the user to a database hosted at : https://coolrpg.000webhostapp.com/

GetDataBase.apk is also an android application and will dsplay the data send by
insertdatabase.apk. 

In the website https://coolrpg.000webhostapp.com/ you can also find the .php files.

This is a simple template to upgrade and insert to others applications

How to install ?
  Just drag and drop the *.apk files into one of your phone folder
